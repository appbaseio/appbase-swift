  • bulk(type:body:headers:completionHandler:)

    Make bulk requests on a specified app or a specific type. Bulk requests can be any of index, update and delete requests.

    For more information : https://www.elastic.co/guide/en/elasticsearch/reference/2.4/docs-bulk.html

    Declaration

    Swift

    public func bulk(type: String? = "_doc", body : [[String : Any]], headers: [String: String]? = nil, completionHandler: @escaping (Any?, Any?, Error?) -> ())

    Parameters

    type

    Type of data that is created in the app (should only be passed if you want to make the request to the that perticular type)
    body

    JSON structured data parameter that has to be passed for updating, Note: For updating data, the JSON must be of the format [[String:Any]]. A quick example of bulk request is :

        let bulkParameters:[[String:Any]] = [
            [ "index": [ "_type": "SwiftClientES"] ],
            ["Title" : "New Movie 4" , "Year" : "2016"],
            [ "delete" : ["_id": "testID"]]
    ]
    header

    The additional headers which can be provided if required

    Return Value

    Received data and response in JSON format and the error occured if any in format (Any?, Any?, Error?)
